# OCaml PI Calc

## About

πの近似値を連分数のアルゴリズムによって再帰的に求める。
OCamlの練習。

## Install

clone this repo into your environment.

## Environment

- OCaml 4.05.0
- Windows Subsystem for Linux

## Usage

```bash
ocaml pi.ml <num>
```

`num`: accuracy of calculation

## Example

```bash
$ ocaml pi.ml 10
 1: 2.000000
 2: 3.500000
 3: 3.066667
 4: 3.156463
 5: 3.138798
 6: 3.142105
 7: 3.141500
 8: 3.141609
 9: 3.141590
10: 3.141593
```
